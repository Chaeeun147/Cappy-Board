# Cappy-Board
This is a project I made through a Stasis Hackclub Starter Project Tutorial. It is a Blinky Board, the first PCB I ever made!
When the PCB is turned on, the LEDs will start to blink!

I made this project because I wanted to start learning hardware, and I thought the Blinky Board in the starter projects would be a great place to start. The guide seemed simple enough, and it would be a great starter project. I also always wanted to try to make my own PCB, but never got a chance to, but I finally got a chance to now!!
